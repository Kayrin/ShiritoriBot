===== Installer node.js et npm sur son perso [Phoko] =====

- Se placer dans son home :
$ cd ~

- Créer un répertoire appelé "node" pour y mettre node puis y accéder
$ mkdir node
$ cd node

- Télécharger le script pour installer node :
$ wget https://raw.github.com/dmrub/portable-node/master/bin/install-node.sh

- Lui donner les droits d'execution et l'exécuter
$ chmod 700 ./install-node.sh
$ ./install-node.sh

- Ajouter node et npm au PATH :
$ echo 'export PATH=$HOME/node/share/nodejs/node-v6.9.5-linux-x64/bin:$PATH' >> ~/.bashrc
$ source ~/.bashrc


======================================================================
Il faut savoir que les modules npm par défaut sont installés (en même temps que npm et node) dans le répertoire ~/node/share/nodejs/node-v6.9.5-linux-x64/lib/node_modules/npm/node_modules

Lorsque vous allez essayer de lancer un programme ou d'installer un module (avec la commande npm install --save nomdumodule.js), ceux-ci vont s'installer dans le répertoire ~/node/node_modules qui sera le répertoire où node ira chercher les modules. Comprenez donc bien que tous les modules situés dans ~/node/share/nodejs/node-v6.9.5-linux-x64/lib/node_modules/npm/node_modules ne seront PAS reconnus par node !

Si une erreur surgit dans l'exécution d'un script (par exemple "request.js missing" ou encore "inherit.js missing"), essayez de voir s'ils ne sont pas déjà dans le dossier ~/node/share/nodejs/node-v6.9.5-linux-x64/lib/node_modules/npm/node_modules et si c'est le cas, copiez les dans le dossier ~/node/node_modules
S'ils ne sont pas déjà dans le premier dossier, installez-les avec npm install --save nomdumodule.js

======================================================================


Et voilà c'est bon normalement ^^
Pour exécuter un programme, suffit se placer dans le répertoire du .js et taper
$ node fichier.js
